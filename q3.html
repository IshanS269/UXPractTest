Typesetting compared to Desktop Publishing

Old Fashioned Typesetting

Typesetting is the process of arranging text for things like books or websites. It involves organizing letters and symbols so its easy to read. They use things metals like lead to create it.

Desktop Publishing

Desktop publsihing makes it easier to use images and texts. With the usage of computers, it allows you to make designs easier to make. With this you can use different colours, fonts, styles that was harder and more money.

HTML and CSS in Modern Web Design

HTML and CSS are used simply to make websites. It allows you to create menus and add photos a lot eaiser and instead of physically printing it, you can just virtually print it and preview on google.

Comparison

Old typesetting and desktop publishing are mainly for printing things like books or posters. Desktop publishing made this faster and more flexible, but both still deal with fixed layouts. HTML and CSS, used for websites, are different because they work in the digital world. Web design allows for content that can change and update instantly.
